# Synthetic sample bundle

This directory contains **toy, synthetic inputs** that mimic the file layouts expected by the repository.

It is meant for:
- checking column names,
- exercising preprocessing scripts,
- understanding how city-to-region reconciliation works,
- validating that the repository can be shared publicly without bundling full third-party datasets.

It is **not** meant for training claims, benchmarking, or reproducing the paper’s results.
